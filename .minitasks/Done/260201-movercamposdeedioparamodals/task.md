# Mover campos de edição para modals

Mover campos de edição de instruções de sistema e ferramentas para um dialog dedicado, onde mostre um textarea maior para essa edição.

Campos que possuem hints serão movidos para lá também.

No lugar onde eram os textareas originais, coloque previews ou indicadores destes campos.
